## Ringkasan

Dinding emoji (atau bisa disebut juga "tayangan emoji dalam layar") mengambil emoji dan emoji khusus yang dikirim di obrolan penayang di Owncast dan menayangkannya di layar.


**Versi terkirim:** 1.1.4~ynh2
